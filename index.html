Information

The video pointer is used to call out specific locations at specific points in time in a video. It makes use of the canvas element to generate the lines pointing to the appropriate video location.

Initialization

$('#demo-video-pointer').videopointer({points:[{x:47,y:63,color:"#9BC9BD",time:12},{x:50,y:35,color:"#A2B5C4",time:1}],endPoint:{style:"circle"}});

Options

Name Type Default Description
points Array [] The array of points that specifies end point in the video in percentage, the color of the end point, and the time in the video to stop.
videoSelector String "video" The css selector for the video element
itemSelector String ".pointer-item" The css selector for the video element
anchorSelector String ".pointer-anchor" The element that is serving as the starting point for each pointer. This element must be found inside the element selected with the itemSelector. If one is not provided this will automatically default to the itemSelector
line Object See line Reference An object containing properties that control how the line looks.
endPoint Object See endPoint Reference An object containing properties that control how the end point looks.

points

Name Type Default Description
x Number 0 Horizontal location of the end point as a percent of the video container width.
y Number 0 Vertical location of the end point as a percent of the video container height.
color String "#000" The color of the fill for the end point.
time Number 0 The number of seconds into the video that the point references.

line

Name Type Default Description
color String "#fff" The default color for all lines being drawn
width Number 2 The width of the line. Also known as the stroke width.
angle Number 45 If using angled lines this is the angle at wich the line is made. If the angle doesn't work for the space, the line will automatically switch to a 90 degree angle.

endPoint

Name Type Default Description
style String "none" The style of the end point. Current options are "none","circle", and "square"
radius Number 4 The radius of the circle of side length of the square
stroke Number 2 The width of the outline.
strokeColor String "#fff" The default color for the border of the endpoint
fillColor String "#fff" The default color for the fill of the endpoint

Methods

  • rerender
  • clearImages
  • generatePoints
  • destroy

rerender

$('#demo-video-pointer').videopointer('rerender');

Rerender forces a re-rendering of the pointer images. This can be useful if you're changing the elements or their locations on the page.

clearImages

$('#demo-video-pointer').videopointer('clearImages');

Removes all pointer images from the page.

generatePoints

$('#demo-video-pointer').videopointer('generatePoints');

Call to the function that generates the pointer images. The rerender function calls clearImages followed by generatePoints.

destroy

$('#demo-video-pointer').videopointer('destroy');

Removes all generate markup and returns the page to it's state prior to the initialization of the video pointer.

Events

  • There are currently no events, though we're working to add some.

Support

Videos

Currently only the html5 video tag is supported. We're working to add support for Youtube and Vimeo videos.
